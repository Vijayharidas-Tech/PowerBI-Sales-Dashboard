# 📊 Power BI Sales Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze sales performance and provide business insights. The dashboard helps users monitor key performance indicators (KPIs), identify sales trends, and make data-driven decisions.

---

## 🚀 Features

- 📈 Total Sales Analysis
- 💰 Profit Analysis
- 📅 Monthly Sales Trend
- 🌍 Regional Sales Performance
- 📦 Product Category Analysis
- 🎯 KPI Cards
- 🎛️ Interactive Slicers and Filters
- 📊 Dynamic Charts and Visualizations

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
